Cofactor er en kundenær rådgiver som bistår med frigjøring av energi, fleksibilitet og kapasitet for det grønne skiftet. Fra idé til gjennomføring. 

Digitale tjenester vi har utviklet for å effektivisere og verdiøke våre rådgivningstjenester.

| Tjeneste | Lenke | Tilgang |
|---|---|---|
| PV produksjon og forbruk | [pv.cofactor.no](https://pv.cofactor.no) | Basisversjon fritt tilgjengelig |
| Områdeanalyser (PV, batteri, vind, forbruk) | [gridload.cofactor.no](https://gridload.cofactor.no) | Intern | 
| Forbruk - kommuner og nettseiere | [elhub.cofactor.no](https://elhub.cofactor.no) | Intern |
