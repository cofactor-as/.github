Cofactors ambisjon som rådgivere er å bidra til grønn næringsutvikling i eksisterende nett. Dagens nettinfrastruktur er kompleks, og analyser krever ofte store datamengder fra ulike kilder – Elhub, sol, vind og batteri. For å legge til rette for nytt forbruk og utnytte dagens nettkapasitet bedre har vi utviklet digitale verktøy som gir presise beslutningsgrunnlag og muliggjør nytt elektrisk forbruk innenfor dagens nettkapasitet. Sammen med kundene finner vi løsninger som kan gjennomføres raskere.

Digitale tjenester som vi har utviklet for å effektivisere og verdiøke våre rådgivningstjenester:

| Tjeneste | Lenke | Tilgang |
|---|---|---|
| PV produksjon og forbruk | [pv.cofactor.no](https://pv.cofactor.no) | Basisversjon fritt tilgjengelig |
| Områdeanalyser (PV, batteri, vind, forbruk) | [gridload.cofactor.no](https://gridload.cofactor.no) | Intern | 
| Forbruk - kommuner og nettseiere | [elhub.cofactor.no](https://elhub.cofactor.no) | Intern |

Våre digitale tjenester er integrert mot Elhub for automatisert innhenting av forbruksdata. Vi har utviklet egne databaser og integrasjoner mot relevante API-er og systemer for å levere helhetlige analyser av energibruk i kombinasjon med solceller, vind og batteri.
